# Security Policy

## Supported versions

The most recent release of this widget is the supported version. Fixes are
applied to the current line and shipped as a new release.

## Reporting a vulnerability

Please do not open a public issue for a security vulnerability.

Report it privately using GitHub's "Report a vulnerability" feature on the
Security tab of this repository, or contact the City of Grand Junction GIS team.
Include the affected version, a description of the issue, and steps to reproduce
it if you have them.

You can expect an acknowledgement within a few business days. Once the report is
confirmed, a fix will be prepared and released, and the report will be credited
unless you ask otherwise.

## Scope

This widget is a derivative work that vendors portions of Esri's ArcGIS
Experience Builder Add Data and Edit widgets. A vulnerability in the original
Esri code, or in Experience Builder itself, should be reported to Esri. Reports
here should concern the code and configuration added by the City of Grand
Junction, CO.
